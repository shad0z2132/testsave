# SavePoint

Your checkpoint for Solana games.

A curated discovery platform for Solana-based web3 games. Find legitimate, playable games while avoiding rugs and vaporware.

## Tech Stack

- Next.js 14 (Pages Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Features

- Curated Solana game directory
- Trending / New / Saved tabs
- Search and filter by genre/status
- Game detail side panel
- Safety score badges
- Save games to personal watchlist
- Community game submissions
- Anonymous voting on upcoming listings
- Admin review dashboard
